Math

代表数学，是一个工具类，里面都是对数据操作的一些静态方法。

public class Test {
    public static void main(String[] args){
        //取绝对值
        System.out.println(Math.abs(-12));//12
        System.out.println(Math.abs(-3.14));//3.14
        //向上取整
        System.out.println(Math.ceil(4.000001));//5.0
        System.out.println(Math.ceil(4.0));//4.0
        //向下取整
        System.out.println(Math.floor(4.9999));//4.0
        System.out.println(Math.floor(4.0));//4.0
        //四舍五入
        System.out.println(Math.round(3.49999));//3
        System.out.println(Math.round(3.500001));//4
        //取最大最小值
        System.out.println(Math.max(10,20));//20
        System.out.println(Math.min(10,20));//10
        //取次方
        System.out.println(Math.pow(2,3));//8
        System.out.println(Math.pow(3,2));//9
        //取随机数[0.0,1.0)
        System.out.println(Math.random());
    }
}

System

System当前程序所在系统，也是工具类。

  • 终止当前运行的java虚拟机，参数用作状态码，非零状态表示异常终止。
System.exit(0);//人为终止虚拟机
  • 获取当前系统的时间，返回的是long类型的时间毫秒值，指的是从1970-1-1 0:0:0开始走到此刻的总的毫秒值，1s=1000ms
long time=System.currentTimeMillis();
 System.out.println(time);
for (int i = 0; i < 1000000; i++) {
    System.out.println(i);
}
long time2=System.currentTimeMillis();
System.out.println(time2);
System.out.println((time2-time)/1000.0+"s");

public class Test {
    public static void main(String[] args) throws IOException, InterruptedException {
        Runtime r=Runtime.getRuntime();
        //System类的exit方法调用的也是Runtime对象的，功能相同
        //r.exit(0);
        //获得虚拟机能够使用的处理器数
        System.out.println(r.availableProcessors());//逻辑CPU核心数
        //获取虚拟机的内存总量
        System.out.println(r.totalMemory()/1024/1024+"MB");
        //获取虚拟机的可用内存量
        System.out.println(r.freeMemory()/1024/1024+"MB");
        //启动某个程序，并返回代表该程序的对象
        Process p=r.exec("C:\\Program Files\\Tencent\\QQNT\\QQ.exe");
        //如果为QQ配置过环境变量可以直接启动
        Thread.sleep(5000);//让程序暂停5秒继续往下走
        p.destroy();//销毁关闭程序
    }
}

BigDecimal

用于解决浮点型运算时，出现结果失真的问题。

构造器 说明
public BigDecimal(double val)注：不推荐使用这个 将double转换为BigDecimal
public BigDecimal(String val) 将String转换为BigDecimal 
public class Test {
    public static void main(String[] args) throws IOException, InterruptedException {
       double a = 0.1;
       double b = 0.2;
        System.out.println(a+b);//0.30000000000000004
        System.out.println("---------------");
        /*BigDecimal a1=new BigDecimal(Double.toString(a));
        BigDecimal b1=new BigDecimal(Double.toString(b));*/
        //valueOf转换一个double成BigDecimal
        BigDecimal a1 = BigDecimal.valueOf(a);
        BigDecimal b1 = BigDecimal.valueOf(b);
        BigDecimal c1 = a1.add(b1);//加
        BigDecimal c2 = a1.subtract(b1);//减
        BigDecimal c3 = a1.multiply(b1);//乘
        //除,后面两个参数是精确几位和舍入模式，ctrl+alt+space会提示
        BigDecimal c4 = a1.divide(b1,3, RoundingMode.HALF_UP);
        BigDecimal i=BigDecimal.valueOf(1);
        BigDecimal j=BigDecimal.valueOf(3);
        BigDecimal k = i.divide(j,3, RoundingMode.HALF_UP);//0.333

        //把BigDecimal转换为double类型的数据
        double rs=k.doubleValue();
        System.out.println(rs);
    }
}
