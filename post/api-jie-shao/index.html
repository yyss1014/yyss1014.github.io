API简单介绍

2025-02-24

API(Application Programming Interface：程序编程接口)：JAVA写好的程序，程序员可以直接调用，Java为自己写好的程序提供了程序说明书。
例、使用scanner接收键盘输入的数据

import java.util.Scanner;//导包
public class HelloWorld {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Enter your age: ");
        int age = sc.nextInt();//等待用户输入一个整数，直到用户输入回车键才会拿到数据
        System.out.println("You are " + age + " years old.");
        System.out.println("Enter your name: ");
        String name = sc.next();//等待用户输入一个字符串，直到用户输入回车键才会拿到数据
        System.out.println(name + " ,Welcome");
    }
}

包

包是用来分门别类管理不同程序的，类似文件夹，建包有利于程序的管理和维护。

  1. 同一个包下的程序，可以直接访问。
  2. 访问其他包下的程序，必须导包才可以访问。
  • add unambiguous imports on the fly（自动导包）
  1. 自己的程序中调用Java提供的程序，也需要先导包才可以使用，注意：Java.lang下的程序（String, System）是不需要导包的，可以直接使用。
  2. 访问多个其他包的程序，这些程序命名又一样的情况，默认只能导入一个程序，另一个程序需要带包名或类名来讨论。
