基本的DOS命令

2025-02-22

打开CMD的方式:

  1. 开始+系统+命令提示符
  2. win+R 输入cmd打开控制台
  3. 在任意文件夹下按住shift键+鼠标右键点击，在此处打开命令行窗口
  4. 资源管理器的地址栏，cmd空格加上路径
  5. 管理员运行

常用的DOS命令：

#盘符切换  (需要进入C盘)C:
#查看路径下的所有文件 dir
#切换目录change directory cd /d C:
#切换下一级目录 cd (目录名称)
#返回上一级目录 cd..
#清理屏幕 cls
#退出终端 exit
#查看电脑ip ipconfig
#打开应用
    计算器 calc
    画图 mspaint
    记事本 notepad
#ping命令 ping www.baidu.com
#文件操作
    创建文件 cd>(文件名称)
    删除文件 delete (文件名称)
    创建目录 md (目录名)
    删除目录 rd (目录名)
