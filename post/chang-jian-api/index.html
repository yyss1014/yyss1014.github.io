String

1.String概述

String创建对象的方式：1.Java所有构造器文字都为此类的对象2.调用String类的构造器初始化对象。

构造器 说明
public String() 创建一个空白字符串对象，不包括任何内容
public String(String original) 根据传入的字符串内容，创建字符串对象
public String(char[] chars) 根据传入的字符数组内容，创建字符串对象
public String(byte[] bytes) 根据传入的字节数组内容，创建字符串对象 
public class StringDemo1 {
    public static void main(String[] args) {
        //直接双引号得到字符串对象，封装字符串数据
        String name="ithema";
        System.out.println(name);//name记录的是字符串地址，但字符串本身做了些改进，可以直接字符串输出内容

        //调用构造器初始化字符串
        String rs1=new String();
        System.out.println(rs1);//""

        String rs2=new String("hello");
        System.out.println(rs2);

        char[] chars={'a','b','c'};
        String rs3=new String(chars);
        System.out.println(rs3);

        byte[] bytes={1,2,3,4};
        String rs4=new String(bytes);
        System.out.println(rs4);

    }
}

2.常用方法

  1. 获取字符串的长度
String s="ithema";
System.out.println(s.length());
  1. 提取某个索引位置处的字符
System.out.println(s.charAt(1));//t
 for (int i = 0; i < s.length(); i++) {
     char c = s.charAt(i);
     System.out.print(c);
}
  1. 把字符串转化为数组，再进行遍历
char[] chars=s.toCharArray();
for (int i = 0; i < chars.length; i++) {
    System.out.print(chars[i]);
}
  1. 判断字符串内容，内容一样返回true
String s1=new String("异象回声");
String s2=new String("异象回声");
System.out.println(s1==s2);//false
System.out.println(s1.equals(s2));//true
  1. 忽略大小写比较字符串内容
String c1="34AeFG";
String c2="34aEfg";
System.out.println(c1.equals(c2));
System.out.println(c1.equalsIgnoreCase(c2));
  1. 截取字符串
String s3="多活一会吧小破游";
String rs=s3.substring(0,5);//包前不包后
System.out.println(rs);//多活一会吧
  1. 从当前位置一直截取到字符串末尾
String rs2=s3.substring(5);
System.out.println(rs2);//小破游
  1. 把字符串中的某个内容替换成新内容
String info="这个电影是个垃圾,垃圾电影！";
String rs3=info.replace("垃圾","*");
System.out.println(rs3);//这个电影是个*,*电影！
  1. 判断字符串中是否包含某个关键字
String info2="Java是世界上最好的编程语言之一";
System.out.println(info2.contains("Java"));//true
System.out.println(info2.contains("java"));//false
System.out.println(info2.contains("Java2"));//false
  1. 判断字符串是否以某个字符串开头
String rs4="张三丰";
System.out.println(rs4.startsWith("张"));//true
System.out.println(rs4.startsWith("张三"));//true
System.out.println(rs4.startsWith("张三2"));//false
  1. 把字符串按照某个指定内容分割成多个字符串，放到一个字符串数组返回
String rs5="张无忌，周芷若，赵敏，殷素素";
String[] names=rs5.split("，");
for (int i = 0; i < names.length; i++) {
     System.out.println(names[i]);
}

3. String的注意事项

  • String是不可变对象
String name="黑马";
name+="程序员";
System.out.println(name);//name内容改变了，name指向的地址也改变了

实际上是在堆内存里生成了一个新的字符串对象。

  • 只要是双引号给出的字符串对象，存储在常量池中，而且相同内容只会存储一份。
String str1 = "abc";
String str2 = "abc";
System.out.println(str1 == str2);//true
  • new String创建字符串对象，每次new出来都是一个新对象，存放在堆内存中。
char[] chars = {'a', 'b', 'c', 'd', 'e', 'f'};
String str1 = new String(chars);
String str2 = new String(chars);
System.out.println(str1==str2);//false

String s1 = "abc";//常量池
String s2 = "ab";
String s3 = s2+"c";//运算存放在堆中
System.out.println(s1==s3);//false

编译优化机制

String s1 = "abc";//常量池
String s2 = "a"+"b"+"c";//java存在编译优化机制，运算都是确定值直接转成"abc"
System.out.println(s1==s2);//true

4.String案例

开发登录界面，提示用户通过键盘输入登录名和密码。根据登陆方法返回的认证结果，判断用户是否登录成功，使用循环控制登录界面最多显示3次。

public class StringDemo1 {
    public static void main(String[] args) {
        for (int i=0;i<3;i++) {
            Scanner sc = new Scanner(System.in);
            System.out.println("请输入登录名称：");
            String loginName = sc.next();
            System.out.println("请输入登录密码：");
            String password = sc.next();
            boolean rs=login(loginName,password);
            if(rs){
                System.out.println("恭喜你，欢迎进入系统");
                break;
            }else{
                System.out.println("登录名或密码错误，请您确认");
            }
        }
    }

    public static boolean login(String loginName, String password) {
        String okloginName ="WASD";
        String okpassword ="123456";
        return loginName.equals(okloginName) && password.equals(okpassword);
    }
}

使用String来开发验证码，实现随机产生验证码，每位可以是数字，大小写字母。

public class StringDemo1 {
    public static void main(String[] args) {
        System.out.println(createcode(4));
    }
    public static String createcode(int n) {
        String code="";
        String data="abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
        Random r = new Random();
        for (int i = 0; i < n; i++) {
            int index = r.nextInt(data.length());
            code += data.charAt(index);
        }
        return code;
    }
}

ArrayList

  集合是一种容器，用来装数据，类似于数组。数组定义完成长度就固定了，而集合大小可变，开发中运用更多。集合种类有很多，Arraylist是其中一种。

  • 容器有哪些方法？增删改查。
构造器 描述
ArrayList() 构建一个初始容量为10的空列表
ArrayList(int initialCapacity) 构建具有指定初始容量的空列表
ArrayList(Collection<?extend E>c) 按照集合的迭代器返回的顺序构造一个包含指定集合元素的列表
  1. 创建一个ArrayList的集合对象并添加数据
ArrayList list=new ArrayList();
list.add("a");
list.add(666);
list.add(99.5);
list.add("a");
list.add("java");

ArrayList<String> list=new ArrayList<String>();
ArrayList<String> list1=new ArrayList<>();//jdk1.7以上支持，建议这种写法
list.add("a");
list.add("a");
list.add("java");
System.out.println(list);//[a, a, java]
  1. 在集合某个索引位置添加一个数据
list.add(1,"Mysql");
System.out.println(list);//[a, Mysql, a, java]
  1. 根据索引获取集合中某个索引位置处的值
 String rs=list.get(1);
System.out.println(rs);//Mysql
  1. 获取集合的大小
 System.out.println(list.size());
  1. 根据索引删除集合中某个元素值，会返回被删除的元素值
System.out.println(list.remove(1));//Mysql
System.out.println(list);//[a, a, java]
  1. 直接删除某个元素值，删除成功则会返回true，反之false，默认删除的是第一次出现的值
System.out.println(list.remove("java"));
System.out.println(list);//[a, a]
list.add(1,"html");
System.out.println(list.remove("a"));
System.out.println(list);//[html, a]
  1. 修改某个索引位置的值，修改后会返回原来的值给我们
System.out.println(list.set(1,"aaa"));//a
System.out.println(list);//[html, aaa]

ArrayList应用案例

购物车中存储了以下商品:Java入门，宁夏枸杞，黑枸杞，人字拖，特级枸杞，枸杞子。现在用户不想购买枸杞了，想要批量删除，请完成该需求。

ArrayList<String> list = new ArrayList<>();
list.add("Java入门");
list.add("宁夏枸杞");
list.add("黑枸杞");
list.add("人字拖");
list.add("特级枸杞");
list.add("枸杞子");
System.out.println(list);

方法一：

for (int i = 0; i < list.size(); i++) {
     if(list.get(i).contains("枸杞")) {
         list.remove(i);
         i--;
      }
}

方法二：

for (int i = list.size()-1; i >=0; i--) {
     if(list.get(i).contains("枸杞")) {
         list.remove(i);

模仿外卖系统的商家系统，完成菜品的上架以及菜品信息浏览功能。
封装Food类

public class Food {
    private String name;
    private double price;
    private String desc;

}

Food对象操作方法类

public class FoodOperater {
    private ArrayList<Food> foodList;
    public void addfood(){
        Food f = new Food();
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入菜品名称：");
        f.setName(sc.next());
        System.out.println("请输入菜品价格：");
        f.setPrice(sc.nextDouble());
        System.out.println("请输入菜品描述：");
        f.setName(sc.next());
        foodList.add(f);
        System.out.println("上架成功");
    }
    public void showAllFoods(){
        if(foodList.size()==0){
            System.out.println("什么菜品都没有，先去上架");
            return;
        }
        for (int i = 0; i < foodList.size(); i++) {
            Food f = foodList.get(i);
            System.out.println(f.getName());
            System.out.println(f.getPrice());
            System.out.println(f.getDesc());
            System.out.println("--------------");
        }
    }
    public void start(){
        while (true) {
            System.out.println("请选择功能：");
            System.out.println("1.上架菜品");
            System.out.println("2.展示菜品");
            System.out.println("3.退出");
            Scanner sc = new Scanner(System.in);
            String choice = sc.next();
            switch (choice) {
                case "1":
                    addfood();
                    break;
                case "2":
                    showAllFoods();
                    break;
                case "3":
                    System.out.println("下次再来哦");;
                    return;
                default:
                    System.out.println("你输入的命令不存在");
            }
        }
    }
}

主函数

public class StringDemo1 {
    public static void main(String[] args) {
        FoodOperater operator = new FoodOperater();
        operator.start();
    }
}
