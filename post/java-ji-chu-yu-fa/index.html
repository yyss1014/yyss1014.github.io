变量

数据类型 变量名=数据

&emsp;&emsp;标识符就是名字，如类名，变量名都是标识符。标识符由数字，字母，下划线(_)，和美元符($)组成。要求不能以数字开头，不能用关键字做名字，且区分大小写。
## **二进制、八进制在程序中的写法**
```java
int a1=0B01100001;          //0B或0b开头二进制
int a2=0141;                //0开头八进制
int a3=0XFA;                //0X或0x开头十六进制

数据类型

  • 整型数据
byte a=127;             //byte范围-128~127
short s=13244;          //short 范围-32768~32767
int i=422424;    
long lg=42424224444L;   //整型字面上默认是int类型，如果是long类型需要后面加上L或l
  • 浮点类型
float f=3.14F;       //字面上默认小数是double，如果是float型后面需要加F或f
double d=56.45;
  • char字符型
char ch='a';
char ch2='中';
  • 字符串类型（扩展一种引用数据类型）
string name="张三";

数据类型转换

  1. 类型范围小的变量，可以直接赋值给类型范围大的变量。
byte a=12;
int b =a;

int c=100;
double d=c;    //d=100.0

char ch='a';   //'a'=97=>000000000 01100001 
int i=ch;      //发生自动类型转换=>00000000 00000000 00000000 01100001
  • byte→short→int→long→float→double
  • char→int→long→float→double
  1. 在表达式中，小范围类型的变量，会自动转换为表达式中较大类型的变量。
  • byte、short、char→int→long→float→double
  • 注意：在表达式中，byte、short、char是直接转换成int类型参与运算。
byte a=10;
int b=20;
long c=30;
long rs=a+b+c;
  1. 强制类型转换
int a=20;
byte b=(byte)a;    // ALT +ENTER 强制类型转换

int i=1500;      //00000000 00000000 00000101 11011100
byte b=(byte)i;  //11011100

double d=99.5;
int m=(int)d;     //m=99，丢掉小数部分，保留整数部分

算数运算符

在java中，两个整数相除还是整数。

int i=5;
int j=2;
i/j      //2.5=>2
i*1.0/j  //2.5

使用加符号连接时能算则算，不能算就用加符号连接在一起。

int a=5;
System.out.println("abc"+a);     //abc5
System.out.println(a+5);         //10
System.out.println("abc"+a+'a'); //abc5a
System.out.println(a+'a'+"abc"); //102abc

自增自减非单独使用时，++a先自增再赋值，a++先赋值再自增。

int m=5;
int n=3;
int result=++m - --m + m-- - ++n + n-- + 3;   //6-5+5-4+4+3

扩展赋值运算符底层代码形式：
eg. a+=b → a=(a的类型)(a+b)

逻辑运算符

符号 叫法 运算逻辑
& 逻辑与 前后表达式都为true才为true
| 逻辑或 只要有一个true结果就为true
！ 逻辑非 true变false，false变true
^ 逻辑异或 前后表达式结果相同则为false，前后表达式结果不同则为true
&& 短路与 判断结果与&相同，若左侧表达式为false，右侧不执行
|| 短路或 判断结果与|一样，左边为true，右侧则不执行
