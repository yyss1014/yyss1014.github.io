Java入门 - Hello World

2025-02-23

记事本开发Hello World

  • 第一个Java代码建议用记事本书写，创建一个新记事本，显示隐藏扩展名，文件的后缀改成.java
  • 建议代码文件名全英文，首字母大写，满足驼峰模式
  1. 编写代码
    第一个代码程序如下：
public class HelloWorld{
    public static void main(String[] args){
        System.out.printIn("Hello World");
    }
}

文件名称必须与代码的类名称一致。

  1. 编译代码
    调出命令行窗口在代码文件当前目录下运行。
javac 文件名.java
  1. 运行代码
    接着在命令行窗口运行。
java 文件名
