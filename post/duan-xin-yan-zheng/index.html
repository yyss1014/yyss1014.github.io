分析接口文档，在请求头里我们需要提供appCode，另外在调用接口时还要提供4个查询参数：

  1. 定义实体类SmsProperties，交由IOC容器管理。
@Component
@ConfigurationProperties(prefix = "spring.cloud.aliyun.sms")
@Data
public class SmsProperties {

    private String templateId;
    private String smsSignId;
    private String host;
    private String path;
    private String appcode;
}

  1. 创建一个smsSend类来发送请求，sendCode是黑马点评短信登陆的service层。这段代码其实是有问题的，我把请求参数写在请求体里了，按接口文档应该写在查询参数中，但是能正常调用我就不改了。
@Service
@Slf4j
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements IUserService {

    @Autowired
    private SmsProperties smsProperties;

    private static final ObjectMapper mapper = new ObjectMapper();

    @Autowired
    private RestTemplate restTemplate;

    @Override
    public Result sendCode(String phone, HttpSession session) {
        //1.校验手机号
        if(RegexUtils.isPhoneInvalid(phone)){
            //2.如果不符合，返回错误信息
            return Result.fail("手机号格式错误");
        }
        //3.符合，生成验证码
        String code = RandomUtil.randomNumbers(6);
        //4.发送验证码
        boolean success = smsSend(phone, code);
        if(!success){
            return Result.fail("短信发送失败");
        }

        //5.保存验证码到session
        session.setAttribute("code", code);
        log.debug("发送验证码成功，验证码：{}", code);

        return Result.ok();
    }

    private boolean smsSend(String phone, String code) {

        HttpHeaders headers = new HttpHeaders();
        headers.add("Authorization", "APPCODE " + smsProperties.getAppcode());
        headers.setContentType(MediaType.APPLICATION_FORM_URLENCODED);//发送表单格式

        MultiValueMap<String,String> params = new LinkedMultiValueMap<>();
        params.add("mobile", phone);
        params.add("templateId", smsProperties.getTemplateId());
        params.add("smsSignId", smsProperties.getSmsSignId());
        params.add("param", "**code**:" + code + ",**minute**:5");

        HttpEntity<MultiValueMap<String,String>> request =
                new HttpEntity<>(params, headers);
        String url = smsProperties.getHost() + smsProperties.getPath();
        String result = restTemplate.postForObject(url, request, String.class);
        try {
            SmsDTO smsDTO = mapper.readValue(result, SmsDTO.class);
            return "0".equals(smsDTO.getCode());
        } catch (Exception e) {
            log.error("短信解析失败: {}", result, e);
            return false;
        }
    }
}

RestTemplate 是 Spring 框架中用于发起 HTTP 请求的工具类，通过配置类初始化 RestTemplate：

@Configuration
public class RestTemplateConfig {

    @Bean
    public RestTemplate restTemplate(RestTemplateBuilder builder){
        return builder.build();
    }

}
