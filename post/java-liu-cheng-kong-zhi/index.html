Java流程控制

2025-02-24

分支结构

  1. if 分支
if(条件表达式){
    代码；
}

if(条件表达式){
    代码1；
}else{
    代码2；
}

if(条件表达式){
    代码1；
}else if(条件表达式){
    代码2；
}
···
else{
    代码n；
}
  • ()后不能跟";"，否则{}代码不受if控制。如果if语句的{}中只有一行代码，{}可以省略不写。
  1. switch分支

  先执行表达式的值，再拿着这个值去与case后的值进行匹配。如果全部case后的值不匹配，再执行default模块。

  • 表达式类型只能是byte、short、int、char，JDK5开始支持枚举，JDK7开始支String，不支持double（浮点数不精确）、float、long
  • case后的值不允许重复，只能是字面量，不能是变量。
  • 正常使用switch的时候，不要忘记break，否则会出现穿透现象。不过当存在多个分支的代码时，可以把相同的代码放在一个case块中，其他的case块都通过穿透性穿透到case块执行代码即可，这样可以简化代码。
switch(表达式){
    case值1:
        执行代码...;
        break;
    case值2:
        执行代码...;
        break;
    ...
    case值n-1:
        执行代码...;
        break;
    default:
        执行代码n;
}

循环结构

  1. for 循环
for (int i = 0; i < 3; i++){
    System.out.println("Hello World");
}
  1. while 循环
int i = 0;
while(i < 5){    
    System.out.println("Hello World");
    i++;
}
  1. do-while 循环
int i=0;
do{
    System.out.println("Hello World");
    i++;
}while(i<3);
  1. 死循环
for( ; ; ){
    System.out.println("Hello World");
}
while(true){    
    System.out.println("Hello World");
}
do{
    System.out.println("Hello World");
}while(true);

continue用于跳出当前循环的当次执行，break跳出并结束当前所在循环的执行。

案例

生成随机数，nextInt(10)生成的是0-9之间的随机数。

import java.util.Random;

public class HelloWorld {
    public static void main(String[] args) {
        Random r = new Random();
        for (int i = 0; i < 20; i++) {
            int data=r.nextInt(10);
            System.out.println(data);
        }
    }
}

猜测幸运数字。

import java.util.Random;
import java.util.Scanner;

public class HelloWorld {
    public static void main(String[] args) {
        Random r = new Random();
        int luckyNumber = r.nextInt(100);
        Scanner sc = new Scanner(System.in);
        while (true) {
            System.out.println("请输入你所猜测的数据：");
            int guessNumber = sc.nextInt();
            if (luckyNumber < guessNumber) {
                System.out.println("你输入的数字过大");
            } else if (luckyNumber > guessNumber) {
                System.out.println("你输入的数字过小");
            }else{
                System.out.println("猜测成功了，可以买单了");
                break;
            }
        }
    }
