Java方法

2025-02-27 2 min read # Java

方法是一种语法结构，它可以把一段代码封装成一个功能，以便重复调用。

修饰符 返回值类型 方法名(形参列表){
    方法体代码(需要执行的功能代码)
    return 返回值
}

public class HelloWorld {
    public static void main(String[] args) {
        int rs=sum(10,20);
        System.out.println(rs);
    }
    public static int sum(int a, int b) {
        int c = a + b;
        return c;
    }
}

形参列表可以有多个，甚至可以没有，多个形参必须用“,”隔开，且不能初始化给值。传输实参给方法的形参时，传输的是实参变量中形参的副本，引用类型的参数传输的存储的地址值。

案例

输出一个int类型的数组内容，要求输出格式为：[11,22,33,44,55]

public static void printArray(int[] arr) {
        if (arr==null){
            System.out.println(arr);
            return;
        }
        System.out.print("[");
        for (int i = 0; i < arr.length; i++) {
            System.out.print(i==arr.length-1?arr[i]:arr[i]+",");
        }
        System.out.print("]");
    }

判断两个数组是否相等

 public static boolean equals(int [] arr1, int [] arr2) {
     if(arr1==null && arr2==null){
         return true;
     }
     if (arr1==null||arr2==null) {
         return false;
     }
     if (arr1.length!=arr2.length) {
         return false;
     }
     for (int i = 0; i < arr1.length; i++) {
         if (arr1[i]!=arr2[i]) {
             return false;
         }
     }
     return true;
 }

方法重载

一个类中，只要一些方法名称相同，形参列表不同，那么它们就是方法重载，其他都不管。
return在无返回值的方法中，可以直接跳出并结束当前方法的执行。
